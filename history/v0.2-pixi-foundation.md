# v0.2 Pixi Foundation

**Date:** 2026-05-08

## Goal

Move the current prototype screens to Pixi.js while preserving gameplay flow.

## Scope

- No new battle mechanics.
- No new upgrades.
- No art pass.
- No animation, particles or sound.

## Acceptance

The same prototype loop works through Pixi.js:

```text
menu -> battle -> result -> upgrades -> next battle -> final
```

## Notes

Pixi.js is installed as a local npm dependency and served through Vite. Playwright smoke tests verify the 5-battle prototype loop in Chromium.
