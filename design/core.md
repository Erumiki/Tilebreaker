# Core Gameplay

Стартовый документ для описания основного игрового цикла Tilebreaker.

## Нужно определить

- Цель уровня
- Основной способ управления
- Типы тайлов/блоков
- Поведение шарика, платформы или другого главного инструмента игрока
- Условия победы
- Условия поражения
- Награды и прогрессия между уровнями

## Правило

Все числовые параметры геймплея должны храниться в `configs/`, а не в коде.
