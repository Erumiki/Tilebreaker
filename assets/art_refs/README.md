# Art References

Accepted style references for Tilebreaker art work.

## Primary Reference

- `astral_archive_style_portrait.png` - accepted portrait mobile fake screenshot for the Astral Archive Defense direction. Use this as the main reference for future monster presentation, battle UI mood, capture effects, resource icons, hand/hold treatment and portrait layout polish.

## Secondary Reference

- `astral_archive_style_landscape.png` - secondary wide screenshot reference for desktop composition, broad scene mood and observatory-under-siege atmosphere.

## Usage Rules

- These are mood, composition and rendering references, not gameplay specifications.
- Tile topology remains owned by `assets/tiles_v2/tile_manifest.json` and `configs/game.json`.
- Future monster portraits, level backdrops, UI chrome, buttons, slots, icons, card/shop frames and effects should reference this style before introducing a new visual language.
- Current implementation already routes menu, intro, battle, result and shop presentation through the MVP art manifest where practical; remaining art work should replace or polish those files in place rather than invent a parallel asset language.
- Preserve the core read: defender of a star archive, red/blue solar/lunar wards, monster breach, living-light hearts, star-dust gold and strict readable board-game UI.
- Do not copy any unreadable artifact from the references if it conflicts with the live UI contract or 3x3 tile readability.
